Información – SysPulse

SysPulse es una plataforma web de monitoreo en tiempo real diseñada para supervisar computadores dentro de una red. Centraliza métricas de rendimiento, estado de conexión, ubicación, alertas, gráficas, exportaciones y análisis para facilitar la toma de decisiones.

CPU RAM Disco Mapa de equipos Dashboard Pro Alertas PDF y Excel Español / Inglés

Qué monitorea SysPulse

  • Uso actual y promedio de CPU.
  • Consumo de memoria RAM.
  • Ocupación de disco.
  • Estado del dispositivo: online u offline.
  • Tiempo de actividad o uptime.
  • Procesos con mayor consumo de CPU y RAM.
  • Ubicación registrada o aproximada de cada computador.
  • Historial de métricas para análisis y reportes.

Módulos principales

  • Inicio: vista general y accesos rápidos.
  • PCs: administración y detalle de equipos monitoreados.
  • Dashboard: mapa, salud general, rankings y gráficas.
  • Alertas: eventos por umbrales o desconexión.
  • Configuración: tema, idioma, zona horaria, umbrales y auto-refresh.
  • Exportación: informes PDF y Excel con interpretación automática.

Funciones agregadas

  • Registro de usuarios con correo electrónico y confirmación.
  • Recuperación de contraseña por correo.
  • Tema blanco predeterminado para presentación.
  • Mapa con Google Maps y marcadores por estado.
  • Dashboard Pro con ranking de riesgo.
  • Reportes claros en PDF y Excel con explicación de resultados.
  • Interfaz bilingüe español e inglés.

Cómo funciona internamente

  1. Cada computador ejecuta un agente de monitoreo.
  2. El agente recolecta métricas del sistema.
  3. El agente envía datos periódicamente al servidor.
  4. El servidor almacena y expone la información por API.
  5. El panel web consulta los datos y actualiza la interfaz.
  6. El usuario puede consultar, filtrar, exportar y analizar resultados.

Ayuda – Guía rápida de uso

Esta guía resume cómo operar SysPulse y cómo validar cada función principal del aplicativo.

Monitoreo de equipos

  • Instala y ejecuta el agente en cada computador.
  • Verifica que el token del agente coincida con el token del servidor.
  • Cuando el agente reporta correctamente, el equipo aparece en la pestaña PCs.
  • El estado online/offline depende del último reporte recibido.

Uso del mapa

  • El mapa muestra equipos con coordenadas guardadas.
  • Los marcadores verdes indican equipos online.
  • Los marcadores rojos indican equipos offline.
  • El buscador permite filtrar por nombre, ubicación o sistema operativo.
  • El botón Centrar mapa reorganiza la vista.

Dashboard Pro

  • La salud general combina desconexión y uso de recursos.
  • Los promedios se calculan con el último punto reportado por cada PC.
  • El ranking de riesgo ayuda a priorizar equipos para revisión.
  • Las gráficas permiten comparar CPU, RAM, disco y estado.

Exportación de informes

  • Selecciona un equipo y pulsa Exportar.
  • Escoge PDF o Excel y define el rango de fechas.
  • El informe incluye resumen, métricas, procesos, gráficas e interpretación.
  • La interpretación explica picos de CPU, uso de RAM y ocupación de disco.

Configuración

  • Cambia tema, idioma y zona horaria desde configuración.
  • Ajusta umbrales de CPU, RAM, disco y tiempo offline.
  • El tema blanco es el predeterminado.
  • El idioma puede cambiarse entre español e inglés.

Solución de problemas

  • No aparece un equipo: revisa agente, token y conexión.
  • No aparece en el mapa: valida que tenga coordenadas.
  • El informe sale sin datos: verifica que haya métricas en el rango seleccionado.
  • No ves cambios: reinicia servidor y recarga con Ctrl + F5.
